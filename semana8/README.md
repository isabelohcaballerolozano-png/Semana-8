## Objetivos de la Semana

### 🎯 Objetivo Principal
Comprender los fundamentos de sistemas operativos de código abierto, especialmente **Linux**, y las prácticas básicas de **seguridad digital** necesarias para proteger plataformas tecnológicas participativas.

### 📚 Temas Clave
- Linux
- Terminal
- Código abierto
- Seguridad digital
- Plataformas comunitarias

---

## Preguntas de Selección Múltiple

Responde en el archivo: `preguntas-semana8.txt`

### Pregunta 1: ¿Qué es un sistema operativo?
- A. Un programa que navega en internet
- B. El software que administra el hardware y permite ejecutar programas
- C. Un archivo de texto
- D. Un servidor web

### Pregunta 2: ¿Cuál es un sistema operativo de código abierto?
- A. Ubuntu
- B. Windows
- C. iOS
- D. Chrome

### Pregunta 3: ¿Qué es la terminal en Linux?
- A. Una aplicación para editar imágenes
- B. Una interfaz que permite ejecutar comandos directamente
- C. Un navegador web
- D. Un antivirus

### Pregunta 4: ¿Cuál mejora la seguridad digital?
- A. Compartir contraseñas con todos
- B. Usar contraseñas fuertes y proteger credenciales
- C. Guardar claves en archivos públicos
- D. Desactivar controles de seguridad

## Actividades Prácticas

### ✅ Actividad 1: Crear Estructura de Carpetas
```bash
mkdir semana8
cd semana8
mkdir seguridad
```

### ✅ Actividad 2: Practicar Comandos Básicos
Crea `seguridad/buenas-practicas.txt` con:
- Proteger contraseñas
- No compartir credenciales
- Mantener software actualizado

### ✅ Actividad 3: Información del Sistema
En `sistema.txt` documenta:
- Resultado de `uname -a`
- Resultado de `whoami`
- Utilidad de conocer información del sistema

### ✅ Actividad 4: Seguridad en Plataformas Comunitarias
En `seguridad-plataformas.txt` enumera 5 prácticas:
1. No exponer datos personales
2. Proteger credenciales
3. Usar conexiones seguras
4. Registrar accesos
5. Limitar acceso a información sensible

### ✅ Actividad 5: Reflexión
En `reflexion-semana8.txt` redacta 8-10 líneas sobre:
**¿Por qué la seguridad digital es importante para proteger plataformas tecnológicas que usan comunidades o procesos participativos?**
