## Objetivos de la Semana

### 🎯 Objetivo Principal
Comprender los fundamentos de sistemas operativos de código abierto, especialmente **Linux**, y las prácticas básicas de **seguridad digital** necesarias para proteger plataformas tecnológicas participativas.

### 📚 Temas Clave
- Linux
- Terminal
- Código abierto
- Seguridad digital
- Plataformas comunitarias

## Preguntas de Selección Múltiple

Responde en el archivo: `preguntas-semana8.txt`

### Pregunta 1: ¿Qué es un sistema operativo?
- A. Un programa que navega en internet
- B. El software que administra el hardware y permite ejecutar programas
- C. Un archivo de texto
- D. Un servidor web

### Pregunta 2: ¿Cuál es un sistema operativo de código abierto?
- A. Ubuntu
- B. Windows
- C. iOS
- D. Chrome

### Pregunta 3: ¿Qué es la terminal en Linux?
- A. Una aplicación para editar imágenes
- B. Una interfaz que permite ejecutar comandos directamente
- C. Un navegador web
- D. Un antivirus

### Pregunta 4: ¿Cuál mejora la seguridad digital?
- A. Compartir contraseñas con todos
- B. Usar contraseñas fuertes y proteger credenciales
- C. Guardar claves en archivos públicos
- D. Desactivar controles de seguridad

## Instrucciones de Entrega

### 📝 Paso 1: Crear Archivos Iniciales
```bash
mkdir semana8
cd semana8
touch preguntas-semana8.txt
touch sistema.txt
touch seguridad-plataformas.txt
touch reflexion-semana8.txt
```

### 🖥️ Paso 2: Completar Cada Archivo
1. **preguntas-semana8.txt**: Respuestas de selección múltiple (A, B, C o D)
2. **sistema.txt**: Información de comandos del sistema
3. **seguridad-plataformas.txt**: 5 prácticas de seguridad digital
4. **reflexion-semana8.txt**: Tu reflexión (8-10 líneas)

### 📤 Paso 3: Publicar en GitHub
```bash
cd ../..                    # Vuelve a la raíz del repo
git status                  # Verifica cambios
git add .                   # Agrega todos los archivos
git commit -m "Actividad semana 8"
git push                    # Envía a GitHub
```

### ✔️ Paso 4: Verificar
- Abre tu repositorio en GitHub
- Confirma que aparezca la carpeta `semana8` con todos los archivos
